Ever wanted to attach a read more link to copied content? Now you can, introducing - Read More, Copy Link.

== Description ==

This plugin is for WordPress, it adds a dashboard menu. Once you have activated the plugin it will automatically alter copied text of the main website using settings in the dashboard. Including copy altering for:

* Copyright information
* Restricting amount

== Installation ==

1. Upload the folder `Read More Copy Link` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Change the settings in the dashboard

== Frequently Asked Questions ==
